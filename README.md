# Front-end Applications

## Live demo

https://sreen020.netlify.app/

## Install

***Clone repo***
```
git clone git@github.com:sreen020/frontend-applications.git
```

***navigate in terminal to directory***

```
cd ../../../..
```

***Install project***

```
npm install
```

***run project***
```
npm start
```
